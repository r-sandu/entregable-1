# Entregable - 01

## Descripción
Programa en Java que:
- Obtiene el valor máximo de un arreglo de enteros
- Comprueba si el valor máximo aparece más de una vez
- Calcula la media aritmética del array
- Indica si la media es mayor, menor o igual a 5

El código ha sido refactorizado y documentado con Javadoc

## Estructura del Proyecto

- `src/EstadisticasArray.java` → Código refactorizado y documentado con Javadoc
- `justificacion_refactorizacion.txt` → Documento que explica los cambios realizados
- `README.md` → Este archivo

## Cambios hechos en la refactorización

- Cambio del nombre de la clase a `EstadisticasArray` para reflejar su función
- Nombres de variables descriptivos en lugar de letras sueltas
- Añadidos comentarios explicativos en cada sección del código
- Mejora de los mensajes mostrados por pantalla para mayor claridad
- Documentación con Javadoc para métodos y propósito del programa

## Cómo ejecutar

1. Abrir terminal en la carpeta raíz del proyecto
2. Compilar el archivo
3. Ejecutar el programa
